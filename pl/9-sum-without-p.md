# Задание

`sum/2` без использования `plus/3`

## Решение

**sum(Xs, X)** - истина, если X равен сумме элементов списка Xs

**Xs** - список, каждый элемент которого имеет тип natural_number

**X** - натуральное число `n`, представленное  в виде `s^n(0)`.

    natural_number(0).
    natural_number(s(X)) :- natural_number(X).

    sum([], 0).
    sum([s(X) | Xs], s(Y)) :- sum([X | Xs], Y).
    sum([0 | Xs], Y) :- sum(Xs, Y).

## Тесты

### sum(+, +)

    ?- sum([s(0)], s(0)).
    true.

    ?- sum([s(0), s(s(0))], s(s(s(0)))).
    true.

    ?- sum([s(0), s(s(0))], s(s(0))).
    false.

### sum(+, -)

    ?- sum([], X).
    X = 0.

    ?- sum([s(0)], X).
    X = s(0).

    ?- sum([s(s(0))], X).
    X = s(s(0)).

    ?- sum([s(s(0)), s(0)], X).
    X = s(s(s(0))).

    ?- sum([s(s(0)), s(s(0)), s(0)], X).
    X = s(s(s(s(s(0))))).

### sum(-, +)

    ?- sum(X, s(s(0))).
    X = [s(s(0))];
    X = [s(s(0)), 0];
    X = [s(s(0)), 0, 0];
    X = [s(s(0)), 0, 0, 0];
    X = [s(s(0)), 0, 0, 0, 0];
    X = [s(s(0)), 0, 0, 0, 0, 0].
